# PDF 플래너 병풍 스타일 재설계

**날짜**: 2026-03-25
**작업자**: 박성준
**범위**: PDF 전자 플래너 내부 디자인 전면 교체

---

## 배경

홈페이지가 "고요한 병풍(Serene Byeongpung)" 컨셉으로 재설계됨에 따라, 실제 판매 상품인 PDF 플래너 내부 디자인도 동일 컨셉으로 통일. 기존 핑크계열 고정 팔레트 → 한지 호백 기반 + 각 테마 색상 악센트로 전환.

---

## 변경 범위

- **커버**: 달+별+방사형 그라디언트 → 서예 커버 (추상 브러시 배경 + 계절 픽토그램)
- **콘텐츠 전체**: 고정 색상 C 상수 교체로 자동 반영
- **7개 테마 전체**: 병풍 스타일로 리마스터

---

## 구현 접근법: 점진적 (A)

기존 T/C 분리 아키텍처를 유지하면서 순차적 업데이트.

---

### 3. 커버 디자인 재작성 `drawCover()` — 서예 커버

**구조:**
```
[배경: bgPage 한지 화이트]
[상단 단청 띠: T.coverDeep, 높이 60px, 불투명도 0.15]
[추상 브러시 스트로크: 3-4개 호형 path, T.coverMid, opacity 0.06~0.12]
[중앙 세로 구분선: T.coverDeep, w=1px, 높이 50%]
[제목 블록:
  - "FORTUNE PLANNER" (영문 세리프, 24px, T.coverDeep)
  - opts.name (한글 세리프, 18px, textDark)
  - opts.year (숫자, 48px bold, T.coverMid)
]
[계절 픽토그램 4개: 하단 1/4 영역, 24px 원형, T.coverDeep]
[하단 단청 띠: T.coverDeep, 높이 40px, 불투명도 0.12]
```

### 4. NavBar 미세 조정 `drawNavBar()`

- 그라디언트: `T.navA` → `T.navB` (기존 유지, 색상만 테마에서 변경)
- 텍스트/아이콘: 흰색 유지
